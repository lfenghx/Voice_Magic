# 元视界 AI 妙妙屋—声音魔法 整合包使用说明

## 项目介绍

元视界 AI 妙妙屋—声音魔法是一个基于 AI 技术的声音处理工具，支持音色创造和音色克隆功能。本整合包提供了一个无需安装任何开发环境即可直接使用的版本。

## 快速开始

### 1. 下载整合包

从项目发布页下载最新的整合包压缩文件（通常为 ZIP 格式）。

### 2. 解压文件

将下载的 ZIP 文件解压到您想要安装的目录。

### 3. 配置 API 密钥

1. 打开解压后的文件夹
2. 找到并双击打开 `.env` 文件
3. 在 `DASHSCOPE_API_KEY=` 后面填写您的 API 密钥
4. 保存并关闭文件

### 4. 启动应用

双击 `run.bat` 文件，等待应用启动。

### 5. 使用应用

在浏览器中访问 `http://localhost:8000`，即可开始使用声音魔法功能。

## 配置说明

### .env 文件配置项

```
# DashScope API Key (必填)
```

## 常见问题及解决方案

### 问题 1：双击 run.bat 后没有反应

**解决方案**：

1. 检查是否已安装 `.NET Framework 4.7.2` 或更高版本
2. 检查杀毒软件是否阻止了程序运行
3. 尝试右键点击 `run.bat`，选择 "以管理员身份运行"

### 问题 2：浏览器访问 http://localhost:8000 显示无法访问

**解决方案**：

1. 检查应用是否已成功启动
2. 检查防火墙设置，确保端口 8000 未被阻止
3. 尝试重启应用

### 问题 3：使用功能时提示 API 错误

**解决方案**：

1. 检查 `.env` 文件中的 API 密钥是否正确
2. 检查网络连接是否正常
3. 检查 API 密钥是否有足够的余额
